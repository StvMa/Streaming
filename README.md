# Sistema de Gestion de Streaming en Go

– Etapa 1: Planeación del Software.

Este sistema simula un servicio de streaming con funcionalidades básicas:
- Registrar contenido
- Editar atributos
- Listar contenido
- Filtrar por género, tipo y año
- Reproducción simulada

## Estructura del proyecto
- /content  → Gestión de contenido
- /search   → Filtros y búsquedas
- /player   → Simulación de reproducción
- /user     → Gestión de usuarios

## Ejecución


go run main.go
